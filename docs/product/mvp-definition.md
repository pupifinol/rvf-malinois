# MVP Definition

The first MVP of RVF Malinois includes:

- realtime telemetry monitoring
- multiwell operational dashboard
- well detail screens
- telemetry historian
- alarm visualization
- commissioning workflows
- reusable equipment catalog
- client portal (read-only)
- RVF operations console

The MVP does NOT include:

- remote control
- AI optimization
- predictive maintenance
- mobile native app
- PLC control integration
